Obiettivo Guida
- Usare GitHub per gestire il proprio codice

Prerequisiti
- Creare un account su GitHub: https://github.com/
- Installare Git sul proprio computer: https://git-scm.com/install/

Steps:
1. Introduzione e perchè usare GitHub
2. Creare nuovo repository su GitHub
3. Creare chiave SSH e inizializzarla
4. Inizializzare il repository localmente
5. Salvare il codice su GitHub
6. Comandi Git essenziali
7. Conclusione e prossime guide
